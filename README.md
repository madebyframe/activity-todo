# Laravel 10 + Vite + Vue2 + Vuetify2 

## Requirement
- php 8.1 or above
- node 16 or above

## ⚡️ How to install

1. `composer install`
2. `cp .env.example .env`
5. `php artisan key:generate`   
6. `npm install`
7. `npm run dev`
8. `php artisan migrate`
8. `php artisan serve`

## Notes

- It runs on an SQLite database for ease of setup.



## Features

- Add a task
- Complete a task
- Delete a task
- Filter the list of tasks to either uncompleted, or all
